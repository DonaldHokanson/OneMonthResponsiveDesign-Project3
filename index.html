Key Ingredients to Responsive Design

Responsive design is an idea that was introduced by Ethan Marcotte from a post he made to The List Apart back in 2010. He has since written a book about this topic simply titled "Responsive Web Design".

There are three main ingredients that Marcotte points out in his book that you need in order to have a truly responsive website.

  1. You need to have a flexible grid
  2. You need to have flexible images and media
  3. You need to use media queries

If you don't use all of these when building your website then it's not truly responsive. While I have just discussed media queries here there is plenty more to learn in the rest of our video series. See the packages below!

“

This is our way forward. Rather than tailoring disconnected designs to each of an ever-increasing number of web devices, we can treat them as facets of the same experience. We can design for an optimal viewing experience, but embed standards-based technologies into our designs to make them not only more flexible, but more adaptive to the media that renders them.

In short, we need to practice responsive web design.

Ethan Marcotte
